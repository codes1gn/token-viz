Features

Everything you need to understand your AI costs

token-viz reads the session logs Copilot CLI already emits — no API keys, no intercepting.

🎯

Exact Token Counts

Reads outputTokens from assistant.message events and context snapshots from session.compaction_start — not estimates, not guesses.

🤖

Subagent Attribution

Detects subagent.started/completed spans and attributes every turn inside that window to the spawned agent. Know exactly what each subagent costs.

📚

Skill Injection Tracking

Captures skill.invoked events and estimates the token cost of every skill's content — the silent cost nobody else tracks.

📊

5-Tab Interactive Report

Overview, Context donut, Timeline bars, Components breakdown, and Top Consumers — one offline HTML file. No server. No internet.

💰

14-Model Pricing Table

Built-in prices for Claude Sonnet/Haiku/Opus, GPT-4o, GPT-4.1, and more. Override with your own JSON. Always shows exact vs estimated clearly.

🔒

Zero Dependencies

Pure Python stdlib. Runs anywhere Python 3.8 runs. No API keys, no network access, no telemetry of any kind.
