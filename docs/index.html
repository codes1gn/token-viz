Everything you need to understand your AI costs

🎯

Exact Token Counts

Reads outputTokens directly from assistant.message events and context snapshots from session.compaction_start — not estimates, not guesses.

🤖

Subagent Attribution

Detects subagent.started/completed spans and attributes turns within that window to the subagent. Know exactly how much each spawned agent costs.

📚

Skill Injection Tracking

Captures skill.invoked events and estimates the token cost of every skill's content injection into context — the silent cost nobody talks about.

📊

5-Tab Interactive Report

Overview, Context donut, Timeline bars, Components breakdown, and Top Consumers — all in a single offline HTML file. No server. No internet.

💰

14-Model Pricing Table

Built-in prices for Claude Sonnet/Haiku/Opus, GPT-4o, GPT-4.1, and more. Override with your own pricing file. Always shows exact vs estimated clearly.

🔒

Zero Dependencies

Pure Python stdlib. No pip install besides the package itself. Runs anywhere Python 3.8 runs. No API keys, no network access, no telemetry.

Get started in 30 seconds

$ git clone https://github.com/codes1gn/token-viz
$ cd token-viz && pip install -e .
$ tv sessions # find most expensive sessions
